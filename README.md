# Política de Privacidad

**Última actualización:** 8 de junio de 2026

En **[Nombre de tu Aplicación o Proyecto]**, nos tomamos muy en serio la privacidad de tus datos. Esta Política de Privacidad describe cómo recopilamos, utilizamos y protegemos la información cuando utilizas nuestra plataforma y nuestros servicios de integración con la API de Meta (WhatsApp Business API).

---

## 1. Información que Recopilamos
Para el correcto funcionamiento de nuestras integraciones de mensajería, podemos procesar la siguiente información:
* **Información de Contacto:** Números de teléfono móvil necesarios para el envío y recepción de mensajes a través de WhatsApp.
* **Datos de Mensajería:** El contenido de los mensajes de texto o multimedia enviados o recibidos estrictamente para la ejecución del servicio.
* **Metadatos técnicos:** IDs de mensajes, estados de entrega (enviado, entregado, leído) provistos por la Graph API de Meta.

## 2. Uso de la Información
Utilizamos la información recopilada únicamente para:
* Gestionar, operar y mantener las funciones de envío de mensajes de la aplicación.
* Enviar notificaciones, alertas o respuestas automatizadas solicitadas por el usuario.
* Garantizar la seguridad y prevenir el uso indebido de nuestra plataforma.

**No vendemos, alquilamos ni compartimos tu información personal o números de teléfono con terceros con fines comerciales.**

## 3. Integración con Terceros (Meta / WhatsApp)
Esta aplicación interactúa directamente con la **Graph API de Meta**. El procesamiento de los datos de WhatsApp también está sujeto a las Políticas de Privacidad oficiales de Meta. Te recomendamos revisar sus términos en [Meta Privacy Policy](https://www.facebook.com/privacy/policy/).

## 4. Retención de Datos
Conservamos los datos de los números de teléfono y registros de mensajes únicamente durante el tiempo que sea estrictamente necesario para cumplir con las funciones de la aplicación o para cumplir con obligaciones legales de auditoría.

## 5. Seguridad de los Datos
Implementamos medidas de seguridad técnicas y organizativas estándar de la industria (como cifrado de datos en tránsito y almacenamiento seguro de tokens de acceso) para proteger tu información contra el acceso no autorizado, alteración o destrucción.

## 6. Cambios a esta Política
Nos reservamos el derecho de actualizar esta Política de Privacidad en cualquier momento. Cualquier cambio será publicado en esta misma sección con su respectiva fecha de actualización.
